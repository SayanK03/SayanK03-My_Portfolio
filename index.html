ML Project 1: Fake_News_Prediction

Here I made a Fake News Prediction System by using "Logistic Regressio Model".

I used some Python Library like Pandas, Numpy , Sklearn and nltk.

In this Predictive System I first add some news data in a CSV file format. Then by using this predictive system we can know wheather the news is 'Real' or 'Fake'. If the output comes 0 that means the news is Real and if it comes 1 it means the news is Fake.
